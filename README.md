# 📚 Formulário de Matrícula

Projeto de um formulário de matrícula escolar desenvolvido com HTML e CSS.

O objetivo do projeto foi praticar estruturação de formulários, organização de layout e estilização de componentes de interface.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Git
- GitHub
